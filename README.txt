Para a compilação do programa, basta a utilização do comando "make" quando estiver no terminal no diretório 
em que se encontra os arquivos do programa. Após isso, utilizar o comando "./programa" no terminal e no mesmo diretório
para assim ocorrer a execução. Após isso, haverá um menu em que poderá escolher 4 opções:

- Digitando 1, exibe todo o calendário (gerando o arquivo completo)
- Digitando 2, exibe apenas um dia específico (digita a data que quer e gera o arquivo com os dados)
- Digitando 3, exibe a partir de uma data (digita a data que deseja e gera o arquivo com os dados)
- Digitando 4, saí do programa


Para a compilação dos testes de caixa preta, é necessária a compilação do arquivo "testesCaixaPreta.c" separadamente.

Para os testes de caixa branca, dentro do diretório "src/Testes" há todos os arquivos do programa com os testes inclusos.
Para este, basta usar o comando "make" e executar com os testes.
